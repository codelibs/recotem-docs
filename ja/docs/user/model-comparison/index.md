# モデル比較画面

複数の学習済みモデルを横並びで比較できます:

![モデル比較](./model-comparison.png)

## 比較方法

[モデル一覧画面](../trained-model-list/)で複数のモデルを選択し、**"Compare"** ボタンをクリックすることで比較画面へ移動します。

## 比較できる情報

| 項目 | 説明 |
|---|---|
| **Algorithm** | 使用した推薦アルゴリズム |
| **Score** | チューニング評価指標でのスコア |
| **Parameters** | ハイパーパラメータの詳細 |
| **Training Data** | 使用した学習データ |
| **Created at** | 作成日時 |

どのモデルをデプロイするかを判断する際の参考にご活用ください。
