# プロジェクト選択画面

プロジェクト一覧では、作成済みのプロジェクトを選択することができます:

![プロジェクト一覧](./project-list.png)

プロジェクトを新規作成するには、右上の **"New Project"** ボタンをクリックします。以下の項目を入力してください:

1. **プロジェクト名** — 同一ユーザー内で一意である必要があります
2. **ユーザーカラム名** — 行動データにおけるユーザー ID に相当するカラム名（例: `user_id`）
3. **アイテムカラム名** — 行動データにおけるアイテム ID に相当するカラム名（例: `item_id`）

任意で **タイムスタンプカラム名** を設定できます。設定した場合、[データ分割時](../start-tuning/#_2-split-config-—-データ分割設定)に時系列順の分割が可能になります。

![プロジェクト作成フォーム](./project-create-form.png)

"Create" ボタンをクリックするとプロジェクトが作成され、ダッシュボードへ移動します。
