# モデル学習画面

この画面では、学習データとモデル設定を組み合わせて新しい推薦モデルを作成します。

## 学習データの選択

アップロード済みの学習データを選択します:

![学習データ選択](./start-training.png)

## モデル設定の選択

チューニングジョブによって生成されたモデル設定、または手動で作成したモデル設定を選択します:

![モデル設定選択](./start-training-filled.png)

**"START TRAINING"** をクリックすると学習が開始され、[モデル詳細画面](../trained-model-detail/)へ移動します。

::: tip
モデル設定は[モデル設定管理画面](../model-config/)でも確認・編集できます。
:::
