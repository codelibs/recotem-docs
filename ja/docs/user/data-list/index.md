# データ管理画面

この画面では学習データおよびアイテムメタデータの確認とアップロードを行います:

![データ管理画面](./data-list.png)

## 学習データのアップロード

**"Upload Training Data"** ボタンをクリックすると、アップロードダイアログが表示されます:

![データアップロード](./data-upload-dialog.png)

CSV ファイルを選択してアップロードします。データ形式は以下のようなものを想定しています:

| user_id | item_id | timestamp |
|---------|---------|-----------|
| 1       | 101     | 2024-01-01 |
| 1       | 102     | 2024-01-02 |
| 2       | 101     | 2024-01-03 |

## アイテムメタデータのアップロード

アイテムのメタデータ（カテゴリ、価格など）をアップロードすることで、[モデル詳細画面](../trained-model-detail/)での推薦結果プレビューがより見やすくなります。
**"Upload Item Metadata"** ボタンから同様にアップロードできます。
