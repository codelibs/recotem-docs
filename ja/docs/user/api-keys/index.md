# APIキー管理

推薦 API へのアクセスには、スコープ付き API キーが必要です。

![APIキー一覧](./api-keys.png)

## APIキーの作成

**"Create API Key"** ボタンをクリックするとキー作成フォームが表示されます:

![APIキー作成](./api-key-create.png)

設定項目:
- **名前** — キーの識別名
- **スコープ** — キーに付与する権限（複数選択可）
  - `recommend` — プロジェクトレベルの推薦 API（A/B ルーティング含む）の呼び出し
  - `predict` — モデル直接指定の推薦 API の呼び出し
  - `batch` — バッチ推薦 API の呼び出し
  - `admin` — 管理操作（モデル学習、デプロイ等）
- **有効期限** — キーの有効期限（未設定の場合は無期限）

作成すると API キーが一度だけ表示されます:

![APIキー作成完了](./api-key-created.png)

::: danger
API キーはこのとき以外に表示されません。必ず安全な場所にコピーして保存してください。
:::

## APIキーの使用

作成した API キーは、リクエストヘッダーに以下のように指定します:

```http
X-API-Key: your-api-key-here
```

## 一覧での確認

キー一覧では以下を確認できます:
- **名前** と **スコープ**
- **最終使用日時** — セキュリティ監査に役立ちます
- **有効期限**

不要になったキーは削除してください。
