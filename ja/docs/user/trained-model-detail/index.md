# モデル詳細画面

モデルの設定確認、ダウンロード、および推薦結果のプレビューができます:

![モデル詳細](./trained-model-detail.png)

## モデル情報

使用したモデル設定（アルゴリズム、パラメータ）や学習データ、評価スコアなどが確認できます。

**"DOWNLOAD"** ボタンからモデルファイルをダウンロードできます。

## 推薦結果のプレビュー

**"Preview results"** パネルを開いて **"Sample"** をクリックすると、ランダムなユーザーへの推薦結果が表示されます:

![推薦プレビュー（生データ）](./model-preview-raw.png)

[データ管理画面](../data-list/)でアイテムメタデータをアップロード済みの場合は、メタデータと紐づけてより直感的なプレビューが可能です:

![推薦プレビュー（メタデータあり）](./model-preview-with-metadata.png)
