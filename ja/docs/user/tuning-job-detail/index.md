# チューニングジョブ詳細画面

ジョブの設定確認や進行状況を確認できます。実行中はステータスが **"Running"** と表示され、**"Live"** バッジとともにリアルタイムでログが更新されます:

![ジョブ詳細（実行中）](./tuning-job-detail.png)

## ログの確認

**"Logs"** セクションには、現時点までに完了した試行のログがリアルタイムで表示されます:

![ログ（実行中）](./log-unfinished.png)

## 完了後の結果確認

ジョブが正常に終了すると、ステータスが **"Completed"** に変わり、**"Job Details"** セクションに最良スコアが表示されます:

![チューニング結果（完了）](./result.png)

完了後に確認できる情報:
- **Best Score** — 最良パラメータでの評価スコア
- **Train After: Yes** に設定していた場合、チューニング結果を元にモデルが自動的に学習されます（[モデル一覧](../trained-model-list/)で確認できます）
- 自動学習が無効の場合は **"Train Model"** ボタンが表示され、手動でモデル学習を開始できます
