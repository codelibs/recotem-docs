# ダッシュボード

プロジェクトを選択すると、ダッシュボードが表示されます。プロジェクトの概要統計を一目で確認できます:

![ダッシュボード](./dashboard.png)

## 表示される統計情報

| 統計項目 | 説明 |
|---|---|
| **Training Data** | アップロード済み学習データの件数 |
| **Tuning Jobs** | 実行されたチューニングジョブの件数 |
| **Trained Models** | 作成済み学習済みモデルの件数 |
| **Deployment Slots** | 設定済みデプロイメントスロットの件数 |

ダッシュボードから、左のサイドバーを使って各機能ページへ移動できます。
