# 定期再学習

定期再学習を設定すると、指定したスケジュールで自動的にモデルの再学習が実行されます。

![定期再学習](./retraining.png)

## スケジュールの設定

**"New Schedule"** ボタンからスケジュールを作成します:

![スケジュール作成](./retraining-create.png)

設定項目:
- **Cron Expression** — cron 形式でスケジュールを指定（例: `0 2 * * *` = 毎日午前2時）
- **Retune parameters before training** — 学習前にパラメータチューニングを再実行するかどうか
- **Auto-deploy after training** — 学習完了後に自動デプロイするかどうか

## 実行履歴

スケジュール詳細では、過去の実行履歴（開始時刻、所要時間、ステータス）を確認できます。

::: tip
定期再学習は新しいデータが継続的に蓄積されるシステムで特に有効です。最新データで学習したモデルを定期的に自動デプロイすることで、推薦精度を維持できます。
:::
