# モデル一覧画面

この画面では、学習済みモデルの一覧を確認できます:

![モデル一覧](./trained-model-list.png)

各モデルのステータス（PENDING / RUNNING / COMPLETED / FAILED）、使用したモデル設定名、作成日時が表示されます。

行をクリックすると[モデル詳細画面](../trained-model-detail/)へ移動します。

新しいモデルを学習するには **"Train New Model"** ボタンをクリックし、[モデル学習画面](../start-training/)へ移動します。

複数のモデルを選択して [モデル比較](../model-comparison/)を行うことも可能です。
