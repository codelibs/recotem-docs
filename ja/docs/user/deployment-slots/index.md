# デプロイメントスロット

デプロイメントスロットは、学習済みモデルを推薦 API として公開するためのエンドポイント設定です。

![デプロイメントスロット一覧](./deployment-slots.png)

## スロットの作成

**"New Slot"** ボタンをクリックするとスロット作成ダイアログが開きます:

![スロット作成](./deployment-slot-create.png)

設定項目:
- **名前** — スロットの識別名
- **モデル** — デプロイする学習済みモデル
- **重み (Weight)** — A/B テスト時のトラフィック分配重み

## スロットの使用

デプロイメントスロットを作成すると、以下の 2 種類のエンドポイントでモデルを利用できます:

### モデル直接指定
```http
POST /inference/predict/{model_id}
```

### プロジェクトレベル（A/B ルーティング）
```http
POST /inference/predict/project/{project_id}
```

プロジェクトレベルのエンドポイントでは、設定されたスロットの **Weight** に基づいてリクエストが振り分けられます。これを利用して[A/B テスト](../ab-tests/)を実施できます。

## スロットの有効/無効

スロットはトグルで有効/無効を切り替えられます。無効化したスロットにはトラフィックが振り分けられません。

::: warning
推薦 API の呼び出しには **predict スコープ** の [API キー](../api-keys/)が必要です。
:::
