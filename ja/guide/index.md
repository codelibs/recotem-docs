# 概要

Recotem は **"専門知識のいらないレコメンド・エンジン"** です。ブラウザベースの UI と REST API を通じて、推薦モデルの構築からデプロイ・運用までを一貫して行うことができます。

## 主な機能

- **アルゴリズムの自動チューニング** --- Optuna と irspack を用いて、データに最適な推薦アルゴリズムとハイパーパラメータを自動探索します。
- **モデルの学習** --- チューニング済みパラメータを用いてモデルを学習し、評価指標を確認できます。
- **API キー管理** --- read / predict / admin の 3 種類のスコープを持つ API キーを発行・管理できます。
- **デプロイメントスロット** --- 学習済みモデルをスロットに割り当て、推薦 API としてリアルタイムに配信できます。
- **A/B テストとコンバージョン追跡** --- 複数モデルに重み付けしたスロットを設定し、コンバージョンイベントを記録・分析できます。
- **定期再学習** --- スケジュールを設定して、モデルの自動再学習を行えます。
- **ダッシュボード** --- プロジェクトごとのモデル数・チューニングジョブ数・デプロイ状況などの統計を一覧できます。
- **ユーザー管理** --- 管理者がユーザーの作成・編集・無効化を行えます。

Recotem は GitHub 上で開発されるオープンソースのソフトウェアであり、Apache License 2.0 の元、自由に使用・開発することができます。
