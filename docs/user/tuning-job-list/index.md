# Tuning Job Management View

In this view, you can check the list of tuning jobs in the project:

![Tuning job list](../../../ja/docs/user/tuning-job-list/tuning-job-list.png)

Each job displays its status (PENDING / RUNNING / COMPLETED / FAILED) and start time. Click a row to navigate to the [Job Detail View](../tuning-job-detail/).

To create a new tuning job, click the **"New Tuning Job"** button to go to the [Tuning Configuration View](../start-tuning/).
