# User Management

::: warning
This page is only accessible to users with staff (administrator) privileges.
:::

The user management page allows you to add, remove, and manage privileges for users who can access Recotem:

![User management](../../../ja/docs/user/user-management/user-management.png)

## Adding a User

Click the **"Create User"** button to open the user creation form:

![User creation](../../../ja/docs/user/user-management/user-create.png)

Configuration options:
- **Username** --- Username used for login
- **Email** --- User's email address (optional)
- **Password** --- Initial password
- **Admin** --- When enabled, grants administrator privileges and access to the user management page

## Deleting a User

Select a user from the list and perform the delete operation.

::: warning
Be careful when deleting the administrator user (`admin`). You may lose access to the system.
:::
